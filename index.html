✊ AirFist

AirFist — Gesture File Sharing

A cross-platform file sharing app for Android & Windows with two transfer modes. Gesture Mode uses MediaPipe hand tracking — close fist to grab a file, open palm on another device to receive it. Classic Mode works like a traditional picker. Smart routing uses local P2P on the same Wi-Fi or WebRTC over the internet — files are always encrypted and never stored raw on the server.

✊ Gesture Detection ⚡ P2P Transfer 🔒 E2E Encrypted 📱 Android + Windows
